1. Abrir excel-merger.exe
2. Seleccionar en la ventana emergente los archivos a unir (cuidar que sean las columnas y distribucion sean iguales. El programa organizará las columnas como se encuentren en los archivos originales)
3. El programa está diseñado para archivos de MDM por lo que su prioridad será unir las tabs que tengan el nombre "Entitites". Si no hay una tab con dicho nombre dentro del archivo, el programa buscará unir la primer pestaña en su lugar.
4. Si el directorio "C:/temporary" no existe en el equipo, el programa preguntará al usuario si desea crearlo. Indicar Y/N como corresponda
5. El archivo final se exportará a C:/temporary
